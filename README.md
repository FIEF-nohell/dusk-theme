# Dusk Theme

A refined dark VS Code theme inspired by **Jellyfish**, reworked with a deeper, darker purple atmosphere.

Dusk keeps the calm, distraction-free philosophy of Jellyfish while pushing the palette toward richer shadows, muted violets, and stronger contrast for long coding sessions.
